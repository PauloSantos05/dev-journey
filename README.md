# dev-journey

A public record of my programming journey, built around 
what companies actually hire for in 2026.

## The roadmap

### Phase 1 — The must-haves (months 1–3)
- Python basics — variables, loops, functions, files
- SQL — SELECT, JOIN, GROUP BY, indexes
- Git + GitHub — commits, branches, pull requests
- Linux terminal — files, permissions, SSH, scripts

### Phase 2 — Get hired territory (months 3–6)
- JavaScript + TypeScript — DOM, async, types
- React — components, hooks, state, routing
- REST APIs — HTTP, JSON, FastAPI
- Docker — images, containers, Compose

### Phase 3 — Stand out (months 6–9)
- Cloud — AWS basics, EC2, S3, Lambda
- CI/CD — GitHub Actions, automated deploys
- PostgreSQL + Redis — queries, caching, performance
- Agile + Scrum — sprints, tickets, real team workflow

### Phase 4 — The 2026 edge (months 9–12)
- LLM APIs — OpenAI, Anthropic, prompt engineering
- RAG + LangChain — vector DBs, agents
- Kubernetes — pods, services, scaling
- System design — architecture, interviews

## Status
Currently on: **Phase 1 — Python**
